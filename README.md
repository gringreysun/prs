# PRS — Личная система готовности
PRS (Personal Readiness System) - это открытая система, которая помогает подготовиться к различным жизненным ситуациям шаг за шагом. 
Она показывает, **что**, **когда** и **в каком порядке** стоит подготовить, чтобы не приходилось каждый раз начинать с нуля.

## Что отличает PRS?
Большинство материалов по подготовке отвечают на вопрос **«что купить»**.
PRS отвечает на вопрос **«что действительно стоит подготовить в первую очередь и почему»**.

## Зачем нужна PRS?
Большинство рекомендаций существуют отдельно друг от друга: статьи, видео, форумы и списки вещей.
Из-за этого приходится самостоятельно искать информацию, изучать десятки источников и каждый раз решать, **что важнее**.
В PRS эта информация объединена в единую систему подготовки, где вы проходите готовые маршруты шаг за шагом.

## Почему это работает?
PRS строится по принципу «от наиболее важного к менее важному».
Подготовка начинается с того, что приносит наибольшую практическую пользу уже сегодня.
PRS не стремится собрать всё возможное. 
Её цель — помочь подготовить действительно важное, не тратя время и деньги на второстепенные вещи.
Каждый следующий шаг дополняет предыдущий, постепенно повышая уровень вашей готовности.

## Принципы системы
- максимум результата за минимум времени;
- сначала закрываем самые важные потребности;
- каждый следующий шаг дополняет предыдущий;
- простые решения лучше сложных;
- открытая разработка и постоянное улучшение.
